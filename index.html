ORCO uses NMR chemical shifts to rank plausible amino-acid identities. G is the ranked Goldilocks candidate set; values in parentheses are posterior probabilities.

Basic usage
Input format

(example) (example) (example)

Header: #NUM AA HA CA CB CO N HN. Use 0.00 for a missing shift. AA is a label column: any non-whitespace label is accepted and it is not used for classification.

For example, Owen produces Owen_orco_results.txt.
