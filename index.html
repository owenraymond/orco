ORCO residue-type classifier

ORCO calculates a ranked posterior distribution over the 20 canonical amino-acid residue types from partial, atom-labeled NMR chemical shifts and converts it into an entropy-adaptive Goldilocks candidate set.

Basic usage
Input format

SHIFTY requires NUM and AA columns plus at least one of HA, CA, CB, CO, N, or HN. Use 0.00, ., ?, NA, N/A, NULL, NONE, or - for a missing shift. AA is retained as a label and is not used for classification.

For example, ubiquitin produces ubiquitin_orco_probabilities.txt.

ORCO results

Residue-type classification results
NUM AA Goldilocks set

Values in parentheses are posterior probabilities. The Goldilocks set is ordered from highest to lowest posterior probability.

Each column stacks the posterior probabilities of the residue types retained in that spin system's Goldilocks set. Rank 1 is shown at the top of each stack.

Methods, manuscript, and reference data

Associated manuscript: Owen Raymond and Bo Chen. ORCO: An Entropy Based Adaptive Residue-Type Classifier for Partial Protein NMR Chemical Shift Observations.

Reference data: Zhang H, Neal S, Wishart DS. RefDB: A database of uniformly referenced protein chemical shifts. J. Biomol. NMR 25:173–195 (2003). ORCO uses the published residue-type- and secondary-structure-specific means and standard deviations.

RefDB does not provide a proline amide-N distribution. ORCO uses μ = 135.627 ppm, σ = 5.150 ppm, and N = 2819 for proline amide N.
